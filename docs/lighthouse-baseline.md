## Known Limitations / Next Steps

| Item | Note |
|---|---|
| **Real-user data** | Connect Sentry Performance or Vercel Analytics for field data (CrUX) |
| **API cold-start** | Render free tier spins down after 15 min inactivity — first load LCP may spike. Upgrade to Starter plan or add a keep-alive ping |
| **Image optimisation** | No images in current build. Once real assets are added, ensure WebP format and lazy-loading |
| **Font loading** | Inter font loaded via CDN in `index.css` — add `font-display: swap` to avoid invisible text |
| **re-audit trigger** | Lighthouse runs automatically on every push to `main` or `staging` via GitHub Actions |
